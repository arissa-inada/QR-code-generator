Link de acesso a aplicação: https://arissa-inada.github.io/QR-code-generator/

Uma aplicação web simples que permite gerar códigos QR diretamente no navegador — sem backend — usando apenas HTML, CSS e JavaScript.
 A ideia é permitir que qualquer pessoa insira um texto ou URL e obtenha um QR Code escaneável de forma rápida e prática.

 Funcionalidades:
 
 .Geração de QR Code em tempo real
 
 .100% client-side (nada é enviado para servidor)
 
 .Compatível com dispositivos móveis
 
 .Suporta texto ou links simples
 
 .Tecnologias

Este projeto foi desenvolvido com:

 HTML – Estrutura da página
 CSS – Estilos e layout
 JavaScript – Geração dinâmica do QR Code

 Como usar:

1.Abra o projeto em um navegador moderno (Chrome, Firefox, Safari etc).

2.Digite um texto ou URL no campo de entrada.

3.Clique no botão de gerar QR Code (ou pressione Enter).

4.O código QR será exibido na tela e pode ser escaneado com qualquer leitor de QR Code de celular.
